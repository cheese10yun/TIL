# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 언어 및 커뮤니케이션

- **기본 응답 언어**: 한국어
- **코드 주석**: 한국어로 작성
- **커밋 메시지**: 한국어로 작성
- **문서화**: 한국어로 작성
- **변수명/함수명**: 영어 (코드 표준 준수)

## 프로젝트 개요

이것은 **TIL(Today I Learned)** 개인 학습 저장소입니다. 다양한 주제의 학습 내용과 코드 예제를 관리합니다.

## 모델 설정

기본 모델: **Haiku 4.5** (글로벌 설정됨 - `~/.claude/settings.json`)

## 저장소 구조

```
TIL/
├── Spring/          # Spring Framework 학습 자료
├── OOP/             # 객체지향 프로그래밍
├── ETC/             # 기타 주제
├── Daily_Dev/       # 일일 개발 기록
└── claude/          # 이 디렉토리 (Claude Code 관련)
```

## 커밋 컨벤션

- 메시지: 한국어, 간결하게 작성
- 예: `내용 추가`, `수정`, `정리`
- 기본 작업: git 커밋 후 자동 푸시
